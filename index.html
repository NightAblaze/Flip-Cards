Rules:

Once the game options have been selected and the game generated players will take turns selecting 2 cards (one after the other).

If the cards' shape matches the player earns 1 point.

If the cards' colour matches the player earns 1 point.

If both the cards' shape and colour match the player will earn 3 points.

If niether the shape or colour match then play proceeds to the next players turn.

If the cards match in any variation then the current player gets another go.

Once there are no more possible matches the game ends and the player with the highest score wins.
